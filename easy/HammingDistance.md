# 问题
汉明距离是两个整数的二进制中对应位置不同字符的个数。
# 代码
```C
int hammingDistance(int x, int y) {
    int z=x^y,a=0;
    for(a=0;z!=0;a++)
    {
        z&=z-1;
    }
return a;
}
```
# 总结
1."^"是异或运算符，算法通过计算x，y异或后其中包含1的个数来计算汉明距离。
2.z&=z-1这个运算的结果就是把z中最后一个1去掉，循环进行运算直到z等于0（所有的1都被去掉）就可以知道z中有多少个1了。
3.也可以使用while，不用for循环。[while(z!=0){a++;z&=z-1}]
