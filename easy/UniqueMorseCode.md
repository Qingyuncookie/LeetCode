# 问题
  摩斯码用 ._ 元素来表示英文26个字母，给你一些单词，判断这些单词中含几种摩斯码形式。
# 代码
```C
int uniqueMorseRepresentations(vector<string>& words)
{
  vector<string> m{".-","-...","-.-.","-..",".","..-.","--.","....","..",".---","-.-",".-..","--","-.","---",".--.","--.-",".-.","...","-","..-","...-",".--","-..-","-.--","--.."};
  unordered_set<string> s;
    for(string x: words)
      {           
         string t="";
         for(char c: x)
            t+=m[c - 'a'];
         s.insert(t);
      }
  return s.size();
 }
```
# 总结
1.unordered_set叫做无序集合容器，是一个存储唯一的关联容器，容器中的元素无秩序关系，允许基于值的快速元素检索，同时支持正向迭代。
2."+="是赋值运算符，a+=b意为a=a+b。
3.注意分号的添加，基础问题。
